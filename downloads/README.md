# Descàrregues

Aquí hi ha l'executable de suport remot:

    OntecQS.exe

El botó "Suport remot" del menú (capçalera i menú mòbil) enllaça a:

    downloads/OntecQS.exe

Si vols substituir-lo, puja el nou .exe amb el mateix nom (OntecQS.exe),
o canvia la constant SUPORT_REMOT_EXE a src/shared.jsx i torna a fer el build
(node build.mjs).
