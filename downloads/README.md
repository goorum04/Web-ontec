# Descàrregues

Col·loca aquí l'executable de suport remot amb aquest nom exacte:

    ontec-suport-remot.exe

El botó "Suport remot" del menú (capçalera i menú mòbil) enllaça a:

    downloads/ontec-suport-remot.exe

Si vols un altre nom o una URL externa, edita la constant SUPORT_REMOT_EXE
a src/shared.jsx i torna a executar el build (node build.mjs).
