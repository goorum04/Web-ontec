# Web Ontec

Sitio web corporativo de **Ontec**: integración de sistemas de videoconferencia,
soluciones audiovisuales y ciberseguridad.

## Contenido destacado

- **Servicios**: videoconferencia, ciberseguridad, integración AV y soporte 24/7.
- **Caso de éxito**: instalación del sistema de videoconferencia en la **Batllia d'Andorra**.
- **Ciberseguridad**: buenas prácticas alineadas con **INCIBE**, con un técnico especializado en colaboración con el organismo.
- **Proceso** de trabajo y formulario de contacto.

## Estructura

```
index.html    Página principal (HTML semántico)
styles.css    Estilos y sistema de diseño
script.js     Interacciones (menú, animaciones, contadores, formulario)
```

## Uso

Es un sitio estático. Basta con abrir `index.html` en el navegador o servirlo:

```bash
python3 -m http.server 8000
# abrir http://localhost:8000
```

No requiere dependencias ni proceso de build.
